ElementPlus 适配器 #

链接 #

  • ElementPlus 官方文档

https://element-plus.org

布局说明 #

由于element-plus的栅栏24列，但在Adminlab布局里统一是12列，所以在适配器里会统一将用户传入的列数乘 2。

就是传参范围请控制在1-12。

表格 columns 说明 #

element-plus里的表格是通过模板的方式写的，比如

<el-table :data="tableData" style="width: 100%">
  <el-table-column prop="date" label="Date" width="180" />
  <el-table-column prop="name" label="Name" width="180" />
  <el-table-column prop="address" label="Address" />
</el-table>

done
 已复制

但是在Adminlab里是通过columns来实现的，就是将写在el-table-column组件上的属性写到 columns 里

那么对应着上面的格式，在Adminlab需要这样写：

  • template
<a-container :columns="columns">
  <el-table></el-table>
</a-container>

    {

      label: "Date",
      width: "180",

    {
      prop: "name",
      label: "Name",
      width: "180",
    },
    {
      prop: "address",
      label: "Address",
    },
  ];
}
