使用 #

import { Adminlab, ElementPlusAdapter } from "adminlab";

app.use(Adminlab, {
  framework: "element-plus",
  adapters: [new ElementPlusAdapter()]
})

布局说明 #

由于element-plus的栅栏24列，但在Adminlab布局里统一是12列，所以在适配器里会统一将用户传入的列数乘 2。

就是传参范围请控制在1-12。

表格 columns 说明 #

element-plus里的表格是通过模板的方式写的，比如

template
script
all
chevron_leftchevron_right
<template>
  <div class="element-demo-1">
    <el-table :data="tableData" v-loading="loading">
      <el-table-column prop="date" label="过期时间" width="180" />
      <el-table-column prop="name" label="用户姓名" width="180" />
      <el-table-column prop="address" label="住址" />

      <el-table-column width="100">
        <template #default="scope">
          <el-button type="text" @click="handleEdit(scope.row)">编辑</el-button>
        </template>
done
已复制

但是在Adminlab里是通过columns来实现的，就是将写在el-table-column组件上的属性写到 columns 里

那么对应着上面的格式，在Adminlab需要这样写：

template
script
all
chevron_leftchevron_right
<template>
  <div class="element-demo-2">
    <a-container :action-get="actionGet" :columns="columns">
      <el-table></el-table>
      <el-pagination></el-pagination>
      <a-form-dialog layout="12"></a-form-dialog>
    </a-container>
  </div>
</template>
done
