AToolbar #

自动生成工具栏，必须在 AContainer 中使用

组件内置支持几种功能的按钮：

  • create

新建按钮，点击后会弹出新建表单弹层

  • refresh

刷新按钮，点击后会刷新当前页面表格的数据

  • search

搜索按钮，点击后会立刻执行action-get方法

  • reset

重置按钮，点击后会重置搜索区表单的值

属性 props #

属性名类型默认值描述
actions
Array<'create' | 'refresh' | 'search' | 'reset'>
undefined

渲染哪些按钮

create
boolean
true

是否显示新建按钮

refresh
boolean
true

是否显示刷新按钮

search
boolean
true

是否显示搜索按钮

reset
boolean
true

是否显示重置按钮

btn-[key]
any
undefined

设置按钮属性，key这里是个泛指，对应UI框架的Button组件属性名，比如btn-size，注意这个设置是针对所有按钮的

事件 events #

属性名类型描述
action
string

如果AToolbarBtn设置的action非内部功能且未注册点击事件，点击后触发该事件

插槽 slots #

属性名描述
default

会将内容渲染到内置按钮的尾部，详细见下方插槽用法示例

插槽用法示例 #

  • 内部组件a-toolbar-btn

a-toolbar-btn有一个action属性，传入内置功能的话比如create会直接唤起新建按钮浮层。详情请前往AToolbarBtn组件查看

  • 任意元素

传入非a-toolbar-btn组件内部将不作任何干涉

all
chevron_leftchevron_right
<template>
  <a-container>
    <a-toolbar :create="false">
      <a-toolbar-btn action="create" color="info">自定义新建</a-toolbar-btn>
      <q-btn>这是个点击啥都不会触发的按钮</q-btn>
      <span>这是一段文本</span>
    </a-toolbar>
  </a-container>
</template>
